# Phishing Awareness Training

## Description

This project is developed to spread awareness about phishing attacks, fake websites, and social engineering tactics. It includes an interactive presentation, quiz section, and a Python-based phishing URL detector.

## Technologies Used

* Python
* PowerPoint
* VS Code

## Features

* Phishing awareness presentation
* Real phishing examples
* Interactive quiz
* URL phishing detector

## Learning Outcome

This project helped in understanding phishing techniques, cyber awareness, and safe online practices.
